## Overview

BuddyPie uses Solidity contracts for onchain delegated budget settlement. These contracts enable users to set spending limits that BuddyPie can settle against.

## BuddyPieDelegatedBudgetSettlement

The main contract handles delegated budget management and USDC settlement.

### Contract Address

| Network | Address |
|---------|---------|
| Base Sepolia | Deploy via script |
| Base Mainnet | Deploy via script |

### Key Features

- Smart account-only settlement authority
- Periodic and fixed budget types
- USDC transfer to treasury
- Budget revocation support

## Deployment

### Prerequisites

1. Install Foundry:
```bash
curl -L https://foundry.paradigm.xyz | bash
foundryup
```

2. Set environment variables:
```bash
export DEPLOYER_PRIVATE_KEY=0x...
export BUDDYPIE_TREASURY=0x...
export BUDDYPIE_USDC=0x036CbD53842c5426634e7929541eC2318f3dCF7e
```

### Deploy to Base Sepolia

```bash
forge script contracts/script/DeployDelegatedBudgetBaseSepolia.s.sol:DeployDelegatedBudgetBaseSepolia \
  --rpc-url https://sepolia.base.org \
  --broadcast
```

### Deploy to Base Mainnet

```bash
forge script contracts/script/DeployDelegatedBudgetBaseMainnet.s.sol:DeployDelegatedBudgetBaseMainnet \
  --rpc-url https://mainnet.base.org \
  --broadcast
```

### Verification

Add verification flags for BaseScan:

```bash
--verify --etherscan-api-key "$ETHERSCAN_API_KEY"
```

## Contract Functions

### Creating Budgets

Budgets are created offchain and registered onchain:

1. Backend generates delegation payload
2. User signs with MetaMask
3. Smart account approves delegation
4. Contract records budget

### Settling Budgets

Only the delegator smart account can trigger settlements:

```solidity
function settle(
    address delegator,
    uint256 budgetId,
    uint256 amount,
    bytes calldata signature
) external;
```

### Revoking Budgets

Users can revoke budgets:

```solidity
function revokeBudget(uint256 budgetId) external;
```

## Budget Types

### Fixed Budget

One-time spending limit that doesn't reset.

```solidity
struct FixedBudget {
    uint256 amount;
    uint256 spent;
    address owner;
    address delegate;
}
```

### Periodic Budget

Resets on interval (day/week/month).

```solidity
struct PeriodicBudget {
    uint256 amount;
    uint256 spent;
    uint256 periodStart;
    uint256 periodDuration;
    address owner;
    address delegate;
}
```

## Integration

### Backend Settlement

The BuddyPie backend triggers settlements:

1. Sandbox operation completes
2. Charge amount calculated
3. Settlement transaction prepared
4. Transaction submitted to contract
5. USDC transferred to treasury

### MetaMask Delegation Toolkit

BuddyPie uses `@metamask/delegation-toolkit` for:

- Creating delegation payloads
- Signing EIP-712 messages
- Managing smart account permissions

### EIP-712 Configuration

For custom USDC tokens, configure in Convex environment:

```bash
X402_EIP712_TOKEN_NAME=USDC
X402_EIP712_TOKEN_VERSION=2
```

## Security Model

### Authority Verification

The contract treats the smart account as the only onchain authority:

- Backend delegate address stored for audit
- Direct settlement from EOA rejected
- All settlements must come through smart account

### Signature Verification

Settlements require valid signatures:

1. Backend signs settlement intent
2. Signature verified against delegate
3. Smart account confirms authorization

### Budget Enforcement

- Cannot exceed configured amount
- Periodic budgets reset automatically
- Revoked budgets reject all settlements
